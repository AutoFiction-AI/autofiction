# Stage 08 — Character & Theme Read

The structure is in place. Now read the manuscript again, end to
end, for **character** and **theme**.

A real editor on this pass asks:

- Is the protagonist a person? Does he develop, or is he a static
  symbol pushed through chapters?
- Does each supporting character carry their own life, or do they
  exist only when the protagonist needs something from them?
- Do the **heritage thread** and the **doubt thread** track across
  the book — not as theme labels announced, but as patterns of
  behavior, speech, ritual, and shame the reader can feel?
- Do the chapters' image systems reinforce each other, or are
  motifs drifting?
- Does the closing question — *can a lie lived long enough become
  true?* — sit inside the protagonist's late-life behavior, or has
  it been left as a thesis statement at the end?

## Inputs

1. `00_shared_aesthetic.md`
2. `blueprint/bible.md`
3. `outline/outline.md`
4. `voice/style_guide.md`
5. Every file in `drafts/v2/` in chapter order.

Read the whole manuscript end to end before writing.

## Required output

Write **one file**: `reads/v2_character_theme_notes.md`. Structure:

### A. Protagonist arc
Two or three paragraphs. Where does the protagonist actually
*change*? Where does the manuscript *say* he changes but the page
does not show it? Where does he get away with being the writer's
mouthpiece? Where does he get away with being too self-aware? Name
the specific chapter beats — by chapter, paragraph or scene — where
the arc holds, and where it slips.

End with a numbered list of arc-repair edits, each tied to a
specific chapter, each one surgical (a scene addition, a cut, a
gesture rewrite, an internal line cut). Do not give vague
instructions ("make the protagonist more conflicted"). Give moves.

### B. Heritage thread
One or two paragraphs on how the Indigenous-heritage thread tracks
across the manuscript as written. Then a numbered list of fixes:
which chapter needs to render heritage that is currently only
named; which chapter is currently leaning on Indigenous reference
as decoration; which chapter is missing a *behavioral* heritage
beat where the bible asked for one. Reference specific beats.

### C. Doubt thread
Same format. Where is the religious doubt rendered in behavior?
Where is it only stated in interior? Where does the manuscript
slip into the bad register where the man's belief-state is
articulated for him by the narrator?

### D. Supporting cast
For each named figure in the dramatis personae: one to three
sentences on how they read across the manuscript. Are they alive?
Are they doing what the bible promised? Are they ever the
chapter's POV when they should be? If a character was promised to
be a full person and reads as functional, name the chapters where
they need real scene time, and what kind of scene would let them
land.

### E. Theme load on the close
One paragraph. Does the closing image carry the premise's question
without speaking it? If the answer is no, what scene work —
gesture, prayer, silence, the way a specific object is handled —
in the final chapter (and the chapter before it) would let the
question land?

End with a numbered list of late-book fixes.

### F. Image system audit
A short table or list: for each motif from the bible, count
roughly how many chapters it appears in, name the chapters where
it lands strongest, and name the chapters where it appears as
decoration without doing work. Recommend cuts and reinforcements.

### G. Per-chapter character/theme notes
Only for chapters that need them. For each:

```
## Chapter N

**Character/theme fixes (numbered, surgical):**
1. ...
2. ...
```

If a chapter is clean at this layer, skip it. Do not pad.

## Voice

Same editorial register as the structural read. Concrete, blunt,
moves rather than gestures.

Write only the notes file.
